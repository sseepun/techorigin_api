# Authentication API

This API set can be used by visitors. To ensure that you are communicating with the API, you will need to include app_id (app.id in your environment file) in JSON Body for every request to the server.

# Sign In

Method Endpoint
POST API_URLapi/auth/signin

Body Variables

Parameter Type Description
app_id String App id
username String User username or email
password String User password
Optional
ip String Visitor IP address for action log
url String Visited url for action log

Response

Parameter Type Description
status Number Response status code
messages Array Response message array
jwt String JSON Web Token used for authorization

Request Example

{
    "app_id": "5789541587749547",
    "username": "SuperAdmin",
    "password": "123456",
    "ip": "127.0.0.1",
    "url": "https://techorigin.co.th/users/signin"
}

Response Example

{
    "status": 200,
    "messages": {
        "success": "คุณได้เข้าสู่ระบบสำเร็จแล้ว"
    },

# Sign Up

Method Endpoint
POST API_URLapi/auth/signup

Body Variables

Parameter Type Description
app_id String App id
firstname String User first name
lastname String User last name
username String User username
email String User email
password String User password
password_confirm String User password confirmation
Optional
ip String Visitor IP address for action log
url String Visited url for action log

Response

Parameter Type Description
status Number Response status code
messages Array Response message array
data Boolean Return true on successful request

Request Example

{
    "app_id": "5789541587749547",
    "firstname": "New",
    "lastname": "User",
    "email": "new@user.com",
    "username": "NewUser",
    "password": "123456",
    "password_confirm": "123456",
    "ip": "127.0.0.1",
    "url": "https://techorigin.co.th/users/signup"
}

Response Example

{
    "status": 200,
    "messages": {
        "success": "คุณได้สมัครสมาชิกสำเร็จแล้ว"
    },
    "data": true
}

# Forget Password

Method Endpoint
POST API_URLapi/auth/forget-password

Body Variables

Parameter Type Description
app_id String App id
email String User email
Optional
ip String Visitor IP address for action log
url String Visited url for action log

Response

Parameter Type Description
status Number Response status code
messages Array Response message array
data array on successful request
user_id String User id
email String User email
salt String Salt variable for resetting password

Request Example

{
    "app_id": "5789541587749547",
    "email": "new@user.com",
    "ip": "127.0.0.1",
    "url": "https://techorigin.co.th/users/forget-password"
}

    "status": 200,
    "messages": {
        "success": "ทำตามขั้นตอนต่อไปเพื่อตั้งรหัสผ่านใหม่"

    "data": {
        "user_id": "11",
        "email": "new@user.com",
        "salt": "UBrf97SgyfFf7cIQtTrVJS90Qg3H7iRMMGPTUtvwfNfRAPTswWjHyAtFxUEjQLEg"
    }
}

# Reset Password Exists

Method Endpoint
GET API_URLapi/auth/reset-password/salt

GET Variables

Parameter Type Description
salt String Salt variable from the response of forgot password API

Body Variables

Parameter Type Description
app_id String App id

Response

Parameter Type Description
status Number Response status code
messages Array Response message array
data array on successful request
id String The unique identifier for this request
user_id String The unique identifier of user associated with salt information
action String Type of action
salt String Salt variable from the response of forgot password API
is_used String Indicate whether this salt is already used or not

Request Example

{
    "app_id": "5789541587749547"
}

Response Example

{
    "status": 200,
    "messages": {
        "success": "คุณสามารถตั้งรหัสผ่านใหม่ได้"
    },
    "data": {
        "id": "4",
        "user_id": "11",
        "action": "RESET PASSWORD",
        "salt": "UBrf97SgyfFf7cIQtTrVJS90Qg3H7iRMMGPTUtvwfNfRAPTswWjHyAtFxUEjQLEg",
        "is_used": "0",
        "ip": "127.0.0.1",
        "used_ip": null,
        "created_at": "2021-01-07 10:05:14",
        "updated_at": "2021-01-07 10:05:14"
    }
}

# Reset Password

Method Endpoint
POST API_URLapi/auth/reset-password

Body Variables

Parameter Type Description
app_id String App id
salt String Salt variable from the response of forgot password API
password_new String New password
password_confirm String Confirm new password
Optional
ip String Visitor IP address for action log
url String Visited url for action log

Response

Parameter Type Description
status Number Response status code
messages Array Response message array
data array on successful request
id String The unique identifier of the user who resets password
email String User email
username String User username

Request Example

{
    "app_id": "5789541587749547",
    "salt": "UBrf97SgyfFf7cIQtTrVJS90Qg3H7iRMMGPTUtvwfNfRAPTswWjHyAtFxUEjQLEg",
    "password_new": "123456",
    "password_confirm": "123456",
    "ip": "127.0.0.1",
    "url": "https://techorigin.co.th/users/reset-password"
}

Response Example

{
    "status": 200,
    "messages": {
        "success": "คุณตั้งรหัสผ่านใหม่ได้สำเร็จแล้ว"
    },
    "data": {
        "id": "11",
        "email": "new@user.com",
        "username": "NewUser"
    }
}
