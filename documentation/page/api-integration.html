# API Integration

# Step 1: Export your users table

Export your users table data from your database in CSV (Comma-separated values) format and import it into the API users accordingly. Please make sure that column names match.

# Step 2: Run User Integration IDs API

Sign in the API framework as a super admin, then run the User Integration IDs API. Necessary data will be provided such as API user ids, emails and reset password salts.

# Step 3: Map user ids

Map the API user ids with your internal user ids. You can map them by checking usernames, emails, Facebook ids or Google ids.

# Step 4: Internal users reset passwords

Your existing users can reset their passwords by using the API reset password salts given in Step 2 and the Reset Password API.
