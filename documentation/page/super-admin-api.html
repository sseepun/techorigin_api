# Super Admin API

This API set can be used by super admins. In order to use this API set, you need to provide JSON Web Token (JWT) in Authorization as Bearer Token type. To ensure that you are communicating with the API, you will need to include app_id (app.id in your environment file) in JSON Body for every request to the server.

# User Type Create

Method Endpoint
POST API_URLapi/sadmin/user-type-create

Body Variables

Parameter Type Description
app_id String App id
parent_id Number of NULL User type parent id
name String User type name
status Number 1 for active and 0 for inactive
Optional
ip String Visitor IP address for action log
url String Visited url for action log

Response

Parameter Type Description
status Number Response status code
messages Array Response message array
data array of the user type created

Request Example

{
    "app_id": "5789541587749547",
    "parent_id": null,
    "name": "New User Type",
    "status": 1,
    "url": "http://techorigin.co.th/sadmin/user-type-create",
    "ip": "127.0.0.1"
}

Response Example

{
    "status": 200,
    "messages": {
        "success": "สร้างข้อมูลสำเร็จ"
    },
    "data": {
        "app_id": "5789541587749547",
