# User API

This API set can be used by members, admins and super admins. In order to use this API set, you need to provide JSON Web Token (JWT) in Authorization as Bearer Token type. To ensure that you are communicating with the API, you will need to include app_id (app.id in your environment file) in JSON Body for every request to the server.
