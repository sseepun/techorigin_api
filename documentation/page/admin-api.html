# Admin API

This API set can be used by admins and super admins. In order to use this API set, you need to provide JSON Web Token (JWT) in Authorization as Bearer Token type. To ensure that you are communicating with the API, you will need to include app_id (app.id in your environment file) in JSON Body for every request to the server.

# User Create

Method Endpoint
POST API_URLapi/admin/user-create

Body Variables

Parameter Type Description
app_id String App id
role_id String User role id
firstname String User first name
lastname String User last name
email String User email
username String User username
password String User password
password_confirm String User password confirmation
Optional
ip String Visitor IP address for action log
url String Visited url for action log

Response

Parameter Type Description
status Number Response status code
messages Array Response message array
data array on successful request
role_id String User role id
firstname String User first name
lastname String User last name
email String User email
username String User username
profile String User profile url path

Request Example

{
    "app_id": "5789541587749547",
    "role_id": 1,
    "firstname": "aaa",
    "lastname": "aaa",
    "email": "aaa@a.com",
    "username": "aaaaa",
    "password": "aaaaaa",
    "password_confirm": "aaaaaa",
    "ip": "127.0.0.1",
    "url": "http://techorigin.co.th/admin/user-create"
}

Response Example

{
    "status": 200,
    "messages": {
        "success": "สร้างข้อมูลสำเร็จ"
    },
    "data": {
        "role_id": "3",
        "firstname": "aaa",
        "lastname": "aaa",
        "email": "aaa@a.com",
        "username": "aaaaa",
        "password": "aaaaaa",
        "profile": null
    }
}

# Traffic Report

Method Endpoint
GET API_URLapi/admin/traffic-report

Body Variables

Parameter Type Description
app_id String App id
type String Report type can be Total Report, Daily Report, Monthly Report, Yearly Report or Full Report
Optional
start_date String Start data of format YYYY-MM-DD
end_date String End data of format YYYY-MM-DD
only_users Boolean Traffic report of users only
only_visitors Boolean Traffic report of visitors only

Response

Parameter Type Description
status Number Response status code
messages Array Response message array
data Array Traffic report array based on type

Request Example

{
    "app_id": "5789541587749547",
    "type": "Daily Report",
    "start_date": "2021-01-09",
    "end_date": "2021-01-11",
    "only_users": false,
    "only_visitors": false
}

Response Example

{
    "status": 200,
    "messages": {
        "success": "ดูข้อมูลสำเร็จ"
    },
    "data": [
        {
            "date": "2021-01-09",
            "total": "1"
        },
        {
            "date": "2021-01-10",
            "total": "2"
        },
        {
            "date": "2021-01-11",
            "total": "1"
        }
    ]
}
